Quick Start Guide

Install the Blender Addon to get started:

  1. Go to Edit > Preferences > Add-ons and click the down arrow ˇ in the upper right.
  2. Click Install from Disk... and select the Folded Paper Engine Blender addon zip file.
  3. Click Install from Disk.
  4. Restart Blender.

Create some ground to stand on:

  1. Once back in your Blender file, add a Mesh Cube to the scene.
  2. Make it a static body for Godot by adding -col to the end of the name.
  3. This is already supported by Godot. See all of the options: Godot node type customization.

Make a basic Player object with Folded Paper Engine:

  1. Create a new mesh object (a sphere is fine).
  2. Select the sphere and press N to open the viewport properties panel.
  3. Go to the Item tab.
  4. Under Folded Paper Engine in the Item tab, check Player.
  5. CONGRATS! You now have a player!

Export a GLB/GLTF for use with Godot:

  1. Go to File > Export and choose glTF.
  2. Check Remember Export Settings. (You'll thank me later 😊)
  3. Click the down arrow ˇ next to Include and check the box that says Custom Properties.
  4. Optionally, check the boxes for Cameras and Punctual Lights if exporting cameras and lights.
  5. Name the file and choose a location inside your Godot project folder.
  6. Click Export.
  7. Your file is now ready to be used by Folded Paper Engine in Godot. See the Godot setup here.
