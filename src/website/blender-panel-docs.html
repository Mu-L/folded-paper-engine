Quick Start Guide

Install the Blender Addon to get started:

Go to `Edit > Preferences > Add-ons` and then click the down arrow `ˇ` in the upper right. Click `Install` and select the `Folded Paper Engine` Blender addon zip file. Then restart Blender.

Create some ground to stand on:

Once back in Blender, add a Mesh Cube to the scene. Make it a static body for Godot by simply adding `-col` to the end of the name. This is something Godot already supports in Blender exports. See all the options here: https://docs.godotengine.org/en/4.5/tutorials/assets_pipeline/importing_3d_scenes/node_type_customization.html

Make a basic player object with Folded Paper Engine:

Make a new mesh object. A sphere should be just fine to start with. Select the Sphere and press `N` to open the viewport properties panel. Make sure you are on the `Item` tab. Under the Folded Paper Engine section in the `Item` tab, check the box that says `Player`. CONGRATS! You now have a player!

Export a GLB/GLTF for use with Godot:

Under `File > Export` choose `glTF`.
