Action maps that keep keyboard, mouse, and controllers in sync

Action map overview

  • Player actions: movement (Forward, Backward, Left, Right), looking around (Turn Up/Down/Left/Right), Run, Jump, Use, and Drop are all mapped so the Player Controls component can read them directly. The action IDs match the ones shipped with the engine so every scene imports ready-to-play. 【F:src/Game/addons/folded_paper_engine/Engine/Input/InputConstants.gd†L62-L75】【F:src/Game/addons/folded_paper_engine/Engine/Features/Object/Character/PlayerControls.gd†L146-L175】
  • UI navigation: UI Up/Down/Left/Right, Page Up/Page Down, Accept, Cancel, and Pause are bundled so menus work with the same devices as gameplay. 【F:src/Game/addons/folded_paper_engine/Engine/Input/InputConstants.gd†L77-L87】

Default bindings

  • PC-friendly defaults ship in the "Default" mapping so WASD, mouse look, and gamepad sticks all drive the same actions without extra setup. You can review or tweak the mappings under addons/folded_paper_engine/Engine/Input/DefaultMappings inside the Godot project. 【F:src/Game/addons/folded_paper_engine/Engine/Input/DefaultMappings/Default.tres†L1-L76】
  • Platform presets are included for Xbox-style, PlayStation-style, and Nintendo-style controllers. Swap mappings or extend them to match your hardware using the provided resource files. 【F:src/Game/addons/folded_paper_engine/Engine/Input/InputConstants.gd†L6-L12】

Device awareness

  • Input is normalized through device proxies so the same action map works across keyboards, mice, and multiple controller families. The engine tracks controller type and axis/button IDs so you can hot-swap devices while testing. 【F:src/Game/addons/folded_paper_engine/Engine/Input/InputConstants.gd†L6-L59】
  • Player Controls automatically reads the current device proxy each physics tick, so you do not need to poll inputs by hand—keep building and let the component translate inputs into movement, camera turning, and Use interactions. 【F:src/Game/addons/folded_paper_engine/Engine/Features/Object/Character/PlayerControls.gd†L145-L175】
