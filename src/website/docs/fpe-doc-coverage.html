Documentation Coverage Map
Where to learn every feature, tailored for minimally technical creators

This page double-checks that every Folded Paper Engine feature called out in our blind-spot review is covered by a friendly guide. Use it as a table of contents and a sanity check before sharing docs with new creators.

Gameplay and interactions

  • Players and Characters: See Objects and Characters for tagging players/NPCs, physics bodies, and cameras.
  • Holdables and Inventory: Inventory and Holdables walks through item kinds, slots, pickup/drop, and UI hooks.
  • Triggers and Commands: Triggers and Commands now includes step-by-step trigger setup, command chaining, and puzzle patterns.
  • Dialogues: Conversations covers speaker setup, dialogue authoring, and reply selection.
  • UI: UI and Cursors explains UI Elements, crosshairs, menu cursor flow, and overlays.

Timing, audio, and feedback

  • Frame Events: Animation and Frame Events shows how to mark frames in Blender and hook them to sounds, doors, or cameras.
  • Audio and Music: Audio explains Speakers, background music, distance falloff, looping, and troubleshooting.

Levels, loading, and scene settings

  • Whole-level transitions: Sub Scenes and Level Loading has door recipes, portal hubs, and unload safety notes.
  • Sub-scenes/streaming: Same guide shows how to nest interiors and pause parents while loaded.
  • Environment metadata: Scene Settings covers sky color, gravity overrides, background music defaults, and FX toggles.

Pipeline and troubleshooting

  • Install and export: Installation and Pipeline and Runtime explain the Blender ↔ Godot loop, importer behaviors, and how to refresh exports safely.
  • Common pitfalls: Troubleshooting lists quick checks for scale, transforms, missing plugins, and stale imports.

Advanced extensions

  • Custom scripts and groups: Objects and Characters highlights ScriptPath and groups so advanced users can extend without breaking the pipeline.
  • Visual overrides: Material and reflective tweaks are documented under Physics and Colliders and Objects and Characters for easy lookup.

If you spot a feature that is not linked here, add a friendly section to the relevant page and update this map so creators always know where to look.
