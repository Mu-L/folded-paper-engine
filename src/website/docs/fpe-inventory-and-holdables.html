Inventory and Holdables
Two separate systems: backpack inventory vs. physics-based holdables

Inventory items (backpack slots)

  1. In Blender, select the item and open the Item properties. Check Trigger if you want entering the trigger to add it to the inventory.
  2. Under the Folded Paper Engine section, enable Inventory Item. Set the Inventory Item Kind ID and Quantity. Duplicate the object if you want multiple pickups.
  3. In Godot, add an FPE Inventory Config node to your scene. In the inspector, add matching Inventory Item Kind entries and set your inventory size. The IDs must match the kinds you set in Blender.
  4. Run the scene: when the player enters the item’s trigger, the quantity is added to the backpack slots. Overflow stays on the trigger so nothing is lost.

Holdable items (physics in your hands)

  1. Select your player in Blender. In Player Controls, choose First Person (optional) and check Can Hold Items. Adjust Hold Zone Distance and Hold Zone Size if needed.
  2. Select the item you want to hold. Make sure it is rigid (has a collider) and, under the Folded Paper Engine tab in the Items section, check Holdable.
  3. Export and run the scene. The player can press the Use key to pick up the holdable. It floats in the Hold Zone and uses physics instead of teleporting.
  4. Press Drop to release it; sway and distance limits prevent the item from stretching too far from the camera.

Using both together

  • Use inventory for collectibles and stackable resources (keys, coins, ammo). Use holdables for physical objects the player can carry in front of them (lamps, boxes, tools).
  • It is fine to keep both features in the same scene: inventory tracks quantities in slots, while holdables are single physics items controlled by the Hold Zone.
  • If you want to remove a holdable after adding it to inventory, run a command on pickup to delete or hide the held object once the quantity is stored.

Troubleshooting

  • Inventory items not appearing: confirm the trigger surrounds the item, the Inventory Config has the matching kind ID, and the inventory size is large enough.
  • Holdables won’t pick up: ensure Can Hold Items is checked on the player and the object is marked Holdable. Items with no collider will not be held.
  • Items wobble or fall: shrink the Hold Zone Size or shorten Hold Zone Distance for a steadier grip.
