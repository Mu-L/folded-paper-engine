Installation and Project Setup
Blender exporter + Godot runtime

Blender add-on

  1. Download the Blender add-on ZIP from the Folded Paper Engine site.
  2. In Blender, open Edit → Preferences → Add-ons, click the upper-right arrow, and choose Install from Disk…. Select the ZIP you downloaded.
  3. Enable the add-on, close Preferences, and press N to confirm you can see the Folded Paper Engine panel in the sidebar.
  4. When you export your scene to GLTF/GLB, open Include and check Custom Properties (plus Cameras and Punctual Lights). Turn on Remember Export Settings so you do not forget this step.
  5. See the Blender panel reference for every property name with the same wording you see in Blender.
  6. Need a quick visual? Watch the Blender setup short for the click-by-click install flow.

Godot plug-in

  1. Download the Godot add-on ZIP from the Folded Paper Engine site.
  2. In Godot, open the Asset Library, click Import, select the add-on ZIP, and click Install.
  3. Open Project → Project Settings → Plugins and enable Folded Paper Engine, then restart Godot once.
  4. Create a new 3D scene and add a Folded Paper Engine node as a child of the root. Add a Sun (DirectionalLight3D) and a WorldEnvironment to finish the starter scene.
  5. In the engine inspector, set Path to your exported .glb/.gltf and assign your WorldEnvironment to Environment. Press play to confirm the player spawns and the mouse is captured.
  6. See the Godot plug-in reference for every inspector field spelled the same way you see it in Godot, plus a feature tour.
  7. Prefer a quick overview? Watch the Godot setup short that walks through import, enabling, and pointing at your GLB.

Project hygiene

  • Keep exported GLB/GLTF files under version control so Godot’s importer can rebuild instances consistently.
  • When updating a GLB, re-export from Blender with transforms applied and let Godot reimport before running.
  • Verify the plug-in is active any time you clone the repo or switch machines; Folded Paper Engine nodes rely on the runtime scripts being enabled.
