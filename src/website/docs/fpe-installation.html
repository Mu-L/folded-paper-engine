Blender add-on (creator-friendly steps)

  1. Download the Folded Paper Engine Blender add-on ZIP from the website.
  2. In Blender, open Edit → Preferences → Add-ons, click the upper-right arrow, and choose Install from Disk…. Pick the ZIP you downloaded.
  3. Enable the add-on so the Folded Paper Engine side panel appears when you press N.
  4. When exporting, choose File → Export → glTF 2.0 and enable Custom Properties, Cameras, and Punctual Lights. Check Remember Export Settings so future exports stay correct.

Godot plug-in

  1. Download the Folded Paper Engine plug-in from the site, then in Godot open the Asset Library → Import tab, select the ZIP, and click Install.
  2. Go to Project → Project Settings → Plugins and enable Folded Paper Engine, then restart Godot.
  3. Create a new 3D scene, add a Folded Paper Engine node, a WorldEnvironment, and a Sun light (the sun keeps level lighting simple).
  4. In the Folded Paper Engine inspector, set Path to your exported .glb/.gltf and assign the Environment to your WorldEnvironment node.
  5. Run the scene: pointer capture, player controls, and behavior registration happen automatically once the engine node is active.

Project hygiene

  • Keep exported GLB/GLTF files in your project folder (under version control if possible) so Godot’s importer can rebuild instances consistently.
  • When updating a GLB, re-export from Blender with transforms applied and let Godot reimport before running.
  • If something feels off, double-check that the plug-in is enabled after switching machines or pulling new code; Folded Paper Engine nodes rely on the runtime scripts being active.
