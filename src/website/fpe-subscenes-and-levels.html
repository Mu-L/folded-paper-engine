Sub Scenes and Level Loading
Level transitions, streaming, and pauses

Whole-level loading

  • Use the LoadLevel command to unload the current GLB and load another. Registered unload procedures run before the scene is cleared.
  • Global registries (speakers, animations, cameras) are reset during unload to prevent cross-level leaks.

Sub-scenes

  • Mark an object as a Sub Scene host with a file path. The host clears its mesh and optionally auto-loads the referenced GLB inside itself.
  • Sub scenes can pause parent activities while loaded and optionally resume when unloaded. An unload delay lets exit fades finish.
  • Commands allow loading/unloading by name or unloading the current sub-scene from within its own context.

Pausing and resuming

  • Activity control utilities can pause all movement (used by sub-scene hosts) and resume when appropriate.
  • Combine with trigger-based level exits to keep player motion disabled while loading screens or cutscenes run.
