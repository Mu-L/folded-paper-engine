Installation and Project Setup
Blender exporter + Godot runtime

Blender add-on

  1. Open Edit → Preferences → Add-ons in Blender.
  2. Click the upper-right arrow and choose Install from Disk…, then select the Folded Paper Engine add-on ZIP.
  3. Enable the add-on and restart Blender to register the FPE property panels.
  4. Enable Custom Properties when exporting to GLB/GLTF so FPE metadata travels with the file.

Godot plug-in

  1. In Godot, open Project → Project Settings → Plugins and enable the Folded Paper Engine plug-in.
  2. Add a WorldEnvironment and a FoldedPaperEngine node to your scene.
  3. In the FoldedPaperEngine inspector, set Path to your exported .glb/.gltf and assign your WorldEnvironment to Environment.
  4. Run the scene: pointer capture, player controls, and feature registration are handled automatically when the engine enters the tree.

Project hygiene

  • Keep exported GLB/GLTF files under version control so Godot’s importer can rebuild instances consistently.
  • When updating a GLB, re-export from Blender with transforms applied and let Godot reimport before running.
  • Verify the plug-in is active any time you clone the repo or switch machines; FPE nodes rely on the runtime scripts being enabled.
